---

#### Default Zoom

The default zoom level when the `world map` is opened.

---

####  Lock Icon Scale

Maintain consistent icon sizes while zooming in/out. To set this per icon, uncheck this option and configure it in the [MapPoint] inspector using. 

 ![](../../../static/img/20260306-180652.png)

---

####  Display Scene Title

Show or hide the title of the current `scene` on the `world map` interface.

---

####  Enable Navigation Path

Enable or disable the [navigation path] system.

---

####  Start Navigation Mouse Button

Define the mouse button used to start [navigation path] by clicking on the map.

---

####  Map Scroll Left/Right/Up/Down Key

Assign keyboard keys for scrolling the map.

---

####  Create Map Marker Key+Mouse Button

Allow players to create `map markers`:

- Hold a specific keyboard key and click with the mouse button to add a `marker`.
- Clicking the same mouse button on an existing `marker` (_without holding the key_) will remove the `marker`.
